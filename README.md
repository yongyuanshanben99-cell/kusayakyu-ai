# 草野球AI査定 Ver.44 キャッシュ対策

## 修正内容
- 閲覧リンクを `viewer.html?v=44#...` の形式で生成
- 古いVer.42の閲覧ページが表示されるキャッシュ問題を修正
- viewer.htmlはService Workerでキャッシュしない仕様へ変更
- 閲覧ページを常に最新版として取得
- 閲覧画面のキャラクター表示を維持
- 投球／打撃ポーズ、左右反転を維持

## 更新後の手順
1. GitHubへ全ファイルを上書き
2. 本体がVer.44になったことを確認
3. 以前の共有リンクは使わず、「閲覧リンク共有」で新しいリンクを作成
4. URLに `viewer.html?v=44` が入っていることを確認

古い共有リンクはVer.42の画面を参照する場合があります。
