# 草野球AI査定 Ver.41 ワンタップ閲覧修正版

## 修正内容
- 閲覧リンクの完全なURLを共有メッセージ本文へ入れる方式に変更
- iPhoneやLINEでURLの `#` 以降が欠ける問題を回避しやすくした
- 「この端末でリンクを確認」ボタンを追加
- viewer.htmlはページ読込時に即座に共有データを確認
- データがない通常のviewer.htmlを開いた場合は原因を明確に表示
- Ver.40リンクとVer.41リンクの両方を読込可能
- 相手は完全な共有URLをタップするだけ
- 閲覧側では編集・追加・削除・保存・AI査定不可

## 正しい共有方法
1. 本体の「閲覧リンク共有」を押す
2. LINEなどの共有画面で、長いURLを含むメッセージを送る
3. 相手はメッセージ内の長いURLをタップする

`viewer.html`だけを送ったり、アドレスバーから短いURLだけをコピーした場合、
データ未読込になります。
