# 草野球AI査定 PWA版

## iPhoneでホーム画面に追加する方法
1. このフォルダをHTTPS対応のWebサーバーへアップロードします。
2. iPhoneのSafariで `index.html` を開きます。
3. 共有ボタンを押します。
4. 「ホーム画面に追加」を選びます。
5. ホーム画面のアイコンから起動すると、アプリ風の全画面表示になります。

## 重要
PWAのインストールとオフライン機能には、原則としてHTTPS配信が必要です。
GitHub Pages、Netlify、Cloudflare Pagesなどの無料サービスで公開できます。

## 機能
- iPhone縦画面専用ゲーム風UI
- 選手一覧・追加・削除
- 選手詳細編集
- 能力値編集
- 簡易AI打撃査定
- メインポジション
- 守備適性A〜G
- 位置ずれを抑えた守備適性図
- 自動保存（localStorage）
- JSON保存・読込
- iOS共有シート
- 印刷画面からPDF保存
- ホーム画面追加
- オフライン起動（Service Worker）

## PDF保存
「PDF」ボタンを押し、iPhoneの印刷プレビューをピンチアウトして共有するとPDFとして保存・送信できます。


## Version 2 変更点

- 走力・肩力・守備力・捕球は手動入力
