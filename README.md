# 草野球AI査定 Ver.35 閲覧パック共有

## Ver.35の変更
- HTML共有を廃止し、専用の `.ksp` 閲覧パックへ変更
- LINE、メール、AirDropなどで共有しやすいファイル形式
- 相手はアプリの「閲覧パックを開く」から `.ksp` を選択
- 閲覧専用モードで選手、能力、成績、守備適性、ランキング、メモを表示
- 閲覧中は選手追加、削除、能力変更、AI査定、保存を禁止
- 「本体へ戻る」で自分の編集データへ復帰
- 閲覧パックを開いても自分の端末内データは変更されません

## iPhoneでの使い方
送信者:
1. 「閲覧パック共有」を押す
2. LINE、メール、AirDropなどで `.ksp` ファイルを送る

受信者:
1. 草野球AI査定 Ver.35を開く
2. 「閲覧パックを開く」を押す
3. 受け取った `.ksp` ファイルを選択

iPhoneのWebアプリでは、ファイルをタップしただけでPWAへ直接関連付けることはできないため、
「閲覧パックを開く」を1回押す方式です。
