# 草野球AI査定 Ver.40 リンク共有完成版

## 新機能
- 「閲覧リンク共有」を追加
- 選手データを圧縮してURLへ埋め込み
- 相手は送られたリンクをタップするだけ
- viewer.htmlがデータを自動展開
- JSONや.kspを選ぶ操作は不要
- 閲覧側では編集・追加・削除・保存・AI査定不可
- 元データには影響なし
- リンクが長すぎる場合は閲覧パック共有へ案内

## 共有方法
1. 本体で「閲覧リンク共有」を押す
2. LINE、メールなどでURLを送る
3. 相手はURLをタップするだけ

## 注意
GitHub Pagesの公開URL上で使用してください。
リンク自体に閲覧データが含まれます。第三者へ転送されると、その人も閲覧できます。
