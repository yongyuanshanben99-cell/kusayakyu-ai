# 草野球AI査定 Ver.46 データ保持・Android対応

## データ保持
Ver.45と同じlocalStorage保存領域をそのまま使用します。
GitHubのファイルを上書きしても、同じURL・同じ端末に保存されているロマンスの選手データは消えません。

## Android対応
- Android Chromeで閲覧可能
- LINEなどのアプリ内ブラウザを検出
- Chromeで開く案内とURLコピーボタンを追加
- 狭いAndroid画面向けに表示を調整
- キャラクター全身表示を維持
- 閲覧リンクは viewer.html?v=46 を使用

## 注意
ホーム画面アプリを削除したり、SafariのWebサイトデータを消すと端末内データも消える場合があります。
更新前にJSONバックアップを保存しておくと安全です。
