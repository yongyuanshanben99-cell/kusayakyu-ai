# 草野球AI査定 Ver.33 ワンタップ閲覧共有

## Ver.33の追加機能
- 閲覧専用HTMLを1ファイルで生成
- 選手データをHTML内へ直接埋め込み
- 相手は受け取ったHTMLを開くだけ
- JSONを選択する操作は不要
- 選手一覧、能力、成績、守備適性、ランキング、メモを閲覧可能
- 選手追加、削除、能力変更、AI査定、保存は不可
- 相手が閲覧しても送信者の元データには影響なし

## 共有方法
1. 本体で「閲覧専用HTML共有」を押す
2. LINE、メール、AirDropなどでHTMLファイルを送る
3. 相手は受け取ったHTMLを開く

LINEで直接開けない場合は、
HTMLを「ファイルに保存」してからSafariで開きます。
