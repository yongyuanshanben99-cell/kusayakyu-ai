# 草野球AI査定 GitHub Pages 公開版 Ver.8

## Ver.8の主な変更
- 選手詳細を本格的な野球ゲーム風UIへ全面刷新
- オリジナルのデフォルメ野球キャラクター
- 大きな頭・大きな目・帽子・ユニフォーム・捕手ミット
- 球場背景、照明、芝生
- 総合評価、信頼度、タイプ、メイン・サブポジション
- 能力ゲージ、特殊能力、成績表のゲームUI化
- SmartScorer PDF読込を4段階ウィザード化
- PDF解析に失敗した場合の「スタッツ文字列貼り付け」機能
- 上部タブと下部ゲームメニュー

## GitHubへの更新方法
1. ZIPを解凍します。
2. GitHubリポジトリで `Add file` → `Upload files` を開きます。
3. 解凍した中身をすべて上書きアップロードします。
4. `Commit changes` を押します。
5. 1〜数分待ち、Safariのタブを閉じて公開URLを開き直します。

ホーム画面版は、古い表示が残る場合に一度削除して再追加してください。
