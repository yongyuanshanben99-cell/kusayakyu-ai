# 草野球AI査定 GitHub Pages 公開版 Ver.12

## 修正内容
- キャラクター画像を `index.html` に直接埋め込み
- `assets` フォルダがなくてもキャラクターが必ず表示
- GitHub Pagesの階層違いによる画像切れを解消
- Ver.12専用キャッシュへ更新

## 更新方法
ZIPを解凍して、GitHubリポジトリへ中身をすべて上書きしてください。

今回は `assets` フォルダはありません。
以下をアップロードします。

- index.html
- manifest.webmanifest
- sw.js
- README.md
- icon-192.png
- icon-512.png
- .nojekyll

画面上部に「Ver.12・画像内蔵」と表示されれば更新成功です。
