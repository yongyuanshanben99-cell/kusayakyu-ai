- 選手詳細をゲーム画面風に全面改修
- キャラクター表示を大型化
- 総合評価カード
- スタイル判定
- 信頼度表示
- メイン・サブポジション表示

- 守備適性図の見た目を強化

## 更新方法
GitHubのリポジトリで `Add file` → `Upload files` を開き、
このフォルダ内のファイルをすべてアップロードしてください。
同名ファイルは上書きされます。

更新後、GitHub Pagesへの反映には1〜数分かかります。
古い表示が残る場合はSafariのタブを閉じて開き直してください。
