# 草野球AI査定 GitHub Pages 公開版 Ver.5

このフォルダの中身を、GitHubリポジトリの一番上へアップロードしてください。

## 上書きするファイル
- index.html
- manifest.webmanifest
- sw.js
- README.md
- icon-192.png
- icon-512.png
- .nojekyll

## 更新方法
GitHubのリポジトリで `Add file` → `Upload files` を開き、
このフォルダ内のファイルをすべて選択してアップロードします。
同名ファイルは上書きされます。

## 更新後
GitHub Pagesの再公開には通常1〜数分かかります。
Safariで古い画面が残る場合は、一度タブを閉じて開き直してください。
ホーム画面版は削除してから再度「ホーム画面に追加」すると、アイコンも更新されます。
