# 草野球AI査定 Ver.32 閲覧専用共有

## 追加機能
- 本体から「閲覧専用データを共有」
- 共有用JSONには表示に必要なデータだけを収録
- `viewer.html`で閲覧専用データを読み込み
- 選手一覧、能力、成績、守備適性、ランキング、メモを閲覧可能
- 選手追加、削除、能力編集、AI査定、保存は不可
- 閲覧者が操作しても送信者の元データには影響しません

## 共有方法
1. 本体で「閲覧専用共有」を押す
2. JSONファイルをLINE、メール、AirDropなどで送る
3. 相手は公開済みの `viewer.html` を開く
4. 「閲覧専用データを開く」からJSONを選ぶ

公開URL例:
`https://ユーザー名.github.io/kusayakyu-ai/viewer.html`

## GitHub更新
