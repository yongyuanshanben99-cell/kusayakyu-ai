# 草野球AI査定 GitHub Pages 公開版 Ver.7

## Ver.7の主な変更
- SmartScorer PDFの読込機能
- チーム名、試合数、勝敗、得点、失点の取得
- 選手別スタッツの一括取得
- 背番号と選手名による既存選手照合
- 新規選手の一括追加
- 既存選手の成績一括更新
- 「上書き・追加」と「全入れ替え」を選択可能
- PDF読込結果のプレビュー
- 手動能力（走力・肩力・守備力・捕球・守備適性）は保持

## 対応PDF
SmartScorer for iOSから出力した、以下のページを含むチームPDFを対象にしています。
- 試合結果一覧
- 集計
- スタッツ

PDF解析にはPDF.jsを使用するため、初回のPDF読込時はインターネット接続が必要です。

## 更新方法
GitHubのリポジトリで `Add file` → `Upload files` を開き、
このフォルダ内のファイルをすべてアップロードしてください。
同名ファイルは上書きされます。
