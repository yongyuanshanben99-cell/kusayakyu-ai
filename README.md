# 草野球AI査定 Ver.20 完成版

新しい土台で作り直した統合版です。

## 主な機能
- 29選手の2026年成績を内蔵
- 選手一覧・選手詳細
- AI打撃査定（ミート・パワー・選球眼）
- 手動能力と保存ボタン
- 保存後に総合値・一覧・ランキングへ反映
- 投げ腕・打ち方による左右反転
- 投球ポーズ・打撃ポーズ切替
- 守備適性A〜G
- チーム内ランキング
- 打率は規定打席到達者だけ表示
- 安打・二塁打・三塁打・本塁打・打点・得点・盗塁などは未到達者も表示
- JSON保存・読込
- 旧版データの自動移行
