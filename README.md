# 草野球AI査定 GitHub Pages 公開版 Ver.13

## 修正内容
- 2026成績を反映した直後にAI査定を自動実行
- ミート・パワー・選球眼を自動更新
- 打率、長打率、四球率、三振率、打席数を査定に使用
- 少ない打席数では能力の振れ幅を抑える補正
- 「全選手をAI一括再査定」ボタンを追加
- 個別の「AI査定を実行」も同じ計算式へ統一
- 走力・肩力・守備力・捕球・守備適性は変更しない
- キャラクター画像のHTML内蔵は維持

## 更新後
画面上部に「Ver.13・AI自動更新」と表示されれば成功です。

更新後は2026データが自動的に再反映され、
全選手のミート・パワー・選球眼も再計算されます。
